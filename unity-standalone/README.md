# مصرف الوحدة — مشروع مستقل

## خطوات الإعداد في Replit جديد

### 1. إنشاء المشروع
- أنشئ Replit جديد من نوع **Node.js**
- ارفع جميع ملفات هذا المجلد إلى المشروع الجديد

### 2. إعداد قاعدة البيانات
- من لوحة Replit اذهب إلى **Secrets** أو **Database**
- أضف متغير `DATABASE_URL` (PostgreSQL connection string)
- **أو** استخدم Replit's built-in PostgreSQL

### 3. تثبيت المكتبات
```bash
npm run setup
```

### 4. تشغيل في وضع التطوير
```bash
npm run dev
```

### 5. النشر للإنتاج
```bash
npm run build
npm start
```
